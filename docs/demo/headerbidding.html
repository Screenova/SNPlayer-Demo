Header bidding simple example

Using Prebid.JS 9.25+

Add the Screenova bidder in your prebid configuration:

      
        ...
        bids: [{
            bidder: 'retailspot',
            params: {
                placement: '6a3a43656ad3c657d3f8cc33',
                videoLoop:true,
                // add snplayer configuration here
            }
        }]
